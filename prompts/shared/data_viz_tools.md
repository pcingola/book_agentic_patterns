## Visualization tools

All plots take an `input_file` and column name parameters, plus optional styling (title, xlabel, ylabel, figsize, palette). Output is saved as a PNG image in the workspace.

Basic: line_plot, bar_plot, scatter_plot, area_plot. Distribution: histogram, box_plot, violin_plot, kde_plot. Categorical: count_plot, pie_chart. Matrix: heatmap, clustermap.
